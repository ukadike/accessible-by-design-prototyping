# Funding Language

Accessibility Audit Lab expands accessibility auditing beyond conventional websites into creative coding, experimental media, educational technology, civic storytelling, and public-interest tools.

The project addresses a gap in current accessibility infrastructure: many automated tools can identify common HTML issues, but fewer tools help artists, educators, journalists, and creative technologists evaluate canvas-based, multimodal, or sensory-rich interfaces.

By combining WCAG 2.2 automated checks with human-review prompts and p5.js-specific guidance, Accessibility Audit Lab helps creators move from awareness to action.
